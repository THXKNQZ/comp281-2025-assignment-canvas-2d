## สรุปงาน 
- วาดฉากทิวทัศน์ 2D ใน `<canvas>` ด้วยคำสั่งพื้นฐานของ Canvas (เส้น/รูปทรง/ไล่สี ฯลฯ)
- **องค์ประกอบที่ต้องมีครบ 7 อย่าง**
  1) ท้องฟ้า 
  2) ภูเขา 
  3) พระอาทิตย์
  4) ท้องนา
  5) ต้นไม้
  6) บ้าน/กระท่อม
  7) แม่น้ำ
